# CreaMVP Website

Sitio principal de CreaMVP y base del CMS editorial del blog.

## Stack

- Astro
- Tailwind CSS
- Supabase
- Vercel

## Desarrollo local

```bash
npm install
npm run dev
```

## Scripts principales

```bash
npm run dev
npm run build
npm run preview
npm run test:unit
npm run test:e2e
npm run test:api
```

## Variables de entorno

Usa `.env.example` como referencia para configurar el entorno local.

## Sitio

- Produccion: `https://creamvp.com`
- CMS: `https://creamvp.com/cms`
